## Project Overview
This is an application for developers to upload their works and showcase what they have been working on .

It is a community-driven platform where developers can connect, share feedback, and interact with one another.

## Features
User registration and authentication

Automatic profile creation upon registration

Profile customization (bio, skills, profile picture)

Project upload and management

Add tags (frameworks, programming languages, etc.) to projects

Upvote and downvote system

Leave feedback and reviews on other developers' projects

Private messaging between users

Search functionality for:

Projects (by title, tags, author username, etc.)

Developers (by username, skills, tags, or project name)

An API Key to connect our backend to the frontend

## Restrictions
Users cannot review or vote on their own projects

Users cannot message themselves

Users can only edit or delete their own projects


## Installation & Setup

# Clone the repository
    git clone <your-repository-url>
    cd <project-folder>

# Create and activate virtual environment
    python -m venv venv
    source venv/bin/activate   # On macOS/Linux
    venv\Scripts\activate      # On Windows

# Install dependencies
    pip install -r requirements.txt

# Apply migrations
    python manage.py migrate

# Run the development server
    python manage.py runserver
